# WhatsDown

WhatsDown is a privacy-friendly web app for analyzing exported WhatsApp chats.

It lets users upload a WhatsApp `.txt` export and view statistics such as message counts, participant activity, word usage, media messages, deleted messages, and group events.

All analysis runs locally in the browser. The uploaded chat file is not sent to a server.

## Features

- Upload exported WhatsApp `.txt` chat files
- Analyze participant activity
- View message, word, media, and deleted message statistics
- Detect group events such as people joining, leaving, being added, being removed, and changing numbers
- Export statistics to PDF or Excel
- Works fully in the browser

## Tech Stack

- React
